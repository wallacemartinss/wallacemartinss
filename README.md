## ⚡ Kronn.io — o que eu construo hoje

> **Gestão de servidores, do seu primeiro VPS ao Kubernetes.**
> Deploy de aplicações **PHP, Node, Python, Go e Docker** sem tocar em SSH — de uma única VPS a clusters em Swarm e Kubernetes.

Uma plataforma na categoria Forge/Ploi, construída para quem cansou de escolher entre *"simples demais para escalar"* e *"complexo demais para começar"*. Provisionamento, deploy, TLS, observabilidade e rollback — no mesmo lugar.
